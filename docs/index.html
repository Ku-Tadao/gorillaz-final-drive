What is this?

Final Drive was a browser-era driving experiment built with Macromedia Director 8.5 and Shockwave 3D. Developed by Zombie Flesh Eaters (designer: Matt Watkins), it placed the Gorillaz into the Geep and dropped the player into a compact 3D world inspired by the 19-2000 video.

This project provides the full game package, helper scripts, and documentation needed to get it running on modern Windows.

Setup

  1. Get the game

    Use a legitimately obtained copy — the original web release, the Phase One: Celebrity Take Down DVD-ROM, or similar.

  2. Drop in the launcher

    Place Play Gorillaz.bat next to gorillaz_final_drive.exe.

  3. Patch IML32

    Run the included PowerShell script against your IML32.dll to fix the Director memory crash:

    powershell -ExecutionPolicy Bypass -File .\tools\patch_iml32.ps1 .\IML32.dll
  4. Keep files together

    The EXE, launcher, DLLs, and xtras/ folder must stay in the same directory.

  5. Launch

    Run Play Gorillaz.bat. It handles TEMP paths, renderer forcing, DEP disable, and CPU affinity pinning.

History

Final Drive shipped when the Gorillaz website was an experimental multimedia playground. The game used a Shockwave 3D toolchain to build a compact driving sim around the Geep and the 19-2000 aesthetic — complete with a launch screen, looping music, selectable resolution and color depth, and a small open map with ramps, roads, and scenery that felt halfway between a tech demo and a proper bonus game.

That browser-plugin era ended when Chrome dropped NPAPI in 2015 and Adobe killed Shockwave entirely in 2019. The game still matters as one of the most tangible Phase 1 interactive projects. This repo exists to make the surviving Windows projector build easier to preserve and run.
